# star-wars-quiz
Star Wars terminal quiz using recursion and the The Star Wars API. 

## Getting Started
### Prerequisites
* Python3
### Installation
* Clone repository

  `$ git clone https://github.com/dominiquecuevas/star-wars-quiz`
  
* Run quiz in terminal

  `$ python3 quiz.py`
